Server Information
#

A vanilla minecraft server for people who want a simpler experience. The modpacks below only add small QOL features and performance fixes. The server uses PurPur Minecraft as the backend.

Installation
#

Whitelist
#

My servers always include a whitelist, in order to join you must submit your account name and UUID to me:

  1. Find account name and inpput into a UUID finder
  2. Add me as a Frend on Discord: Krieger#0144
  3. Send me the account name and UUID
  4. I will message when I have added you
  5. At this time I will send you the login information for the fileshare (described below)

Prerequisites
#

Knowing how to read and follow instructions.

  1. Download and install the latest version of the Prism Launcher
  2. Launch Prism and sign in to your Microsoft account
  3. Download modpack one of the modpack options below. Both are nearly identicle except one has a mod that enables controller input.
    1. Vanilla Plus Controller
    2. Vanilla Plus Normal
  4. Inside Prism click the button in the top left corner that says Add Instance
  5. In the list on the right of the pop-up window click Import
  6. Click the Browse button and navigate to the location of whichever modpack you chose.
  7. Click the instance to launch it, the server should be added already but in the event it is not the address is mc.kriegerhost.xyz

Enjoy!
